1. UserHomeController  모두 구현완료
signup -> 회원가입 컨트롤러
logoutPage -> 로그아웃 컨트롤러
mainPage -> 메인페이지 렌더링하는 컨트롤러


2. PostController
PostsRendering -> 포스트 리스트  (조건별 검색기능을 추가해야한다... -> 아직 구현못함.)
createPostForm -> form dto를 요청
createPost -> Post요청으로 넘어온 데이터를 저장하고, 리다이렉션한다.  -> 여기까지 구현완료
PostDetails -> 포스트 하이퍼 링크로 들어가면, 게시글별 Detail 화면 뿌려줌
updatePost -> 게시글 수정 form dto요청해서 값을 가져온 후 수정진행.
deletePost -> 게시글 삭제
PostSearch -> 검색조건 추가

3. PropertyController
PropertyInfoRendering -> 위치정보와 게시글 정보를 조인해서 가져와야한다. 쿼리 dsl을 사용해야할듯.
PropertyInfoSearch -> 검색조건 추가
